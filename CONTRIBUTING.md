# Contributing to SoulChain

我們非常歡迎您對 SoulChain 專案做出貢獻！

## 如何貢獻

1. Fork 這個倉庫
2. 創建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 開啟一個 Pull Request

## 開發流程

1. 確保您的開發環境已經設置好所有必要的依賴
2. 在開始新功能之前創建一個新分支
3. 編寫代碼時遵循我們的編碼規範
4. 確保所有測試都能通過
5. 更新相關文檔

## 提交規範

- feat: 新功能
- fix: 修復問題
- docs: 文檔修改
- style: 代碼格式修改
- refactor: 代碼重構
- test: 測試用例修改
- chore: 其他修改

## 行為準則

- 尊重所有貢獻者
- 保持專業和友善的交流
- 接受建設性的批評
- 專注於專案目標
